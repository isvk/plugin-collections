# Каталог плагинов для видео-монтажа

## Демо на [pages github](https://isvk.github.io/plugin-collections/build/) или [рабочем сервере](https://isvk.github.io/plugin-collections/build/)

## Постановка задачи

Cделать MVP коммерческого проекта - простая страница с фильтрацией картинок-ссылок по категориям.

## Реализованный функционал

* Слайдер категорий
* Фильтрация картинок-ссылок по категориям
* По скролу страницы происходит подгрузка картинок

## Стек

* React
* Hooks
* Redux
* Saga
* Reselect
* Immutable.js
* React Router
* Styled-components
* GraphQL
* strapi.io

## Back-End

Выполнен самостоятельно на [strapi.io](https://strapi.io/).

На рабочем сервере используюеться strapi и GraphQl запрос, на pages github запрос и картинки замоканы.
